================================================================================
  HOW TO USE
================================================================================

1. Download assets (see above)

2. Start dev server:
   npm run dev

3. Log in to Atlas Access

4. After login, click "3D View" button

5. Drag the card around and watch it swing!

6. Click "2D View" to return to flat badge
